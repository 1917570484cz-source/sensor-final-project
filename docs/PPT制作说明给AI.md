# PPT 制作说明给 AI

请根据以下要求制作课程项目结果展示 PPT。不要重新编造数据，只使用给定图片和数据。

## 1. 基本要求

PPT 主题：

> ESP32-S3 多传感器融合扩展板与精度提升结果展示

整体风格：

- 学术汇报风格，白底或浅灰底，蓝色/黑色作为主色。
- 每页不要堆太多文字，一页一个重点。
- 图片优先放大展示，数据表只保留关键指标。
- 所有图都用 PNG 图片，不要用 SVG。
- 图片路径来自 `ppt_assets/` 文件夹。
- 建议 10-12 页。
